#GIT

git status

git add

git commit -m "[comment]"

git diff HEAD -- [filename]

回到上一次 add或commit的状态(还原)

git checkout -- [filename]

远程仓库添加

git remote add origin [url]

本地内容推送到远程仓库

git push -u origin master

克隆

git clone  [url]
